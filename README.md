A mobile application for smallholder farmers to get crop recommendations and manage farm operations.

## Features

- **Crop Recommendations**: Get personalized crop suggestions based on farm conditions
- **Cultivation Log**: Track farming activities and crop progress
- **Yield Management**: Record and monitor harvest data
- **Sales Tracking**: Manage crop sales and revenue
- **Analytics**: View insights and performance trends
- **Crop Information**: Learn about Malaysian cash crops

## Tech Stack

- **Flutter**: Cross-platform mobile development
- **Firebase**: Authentication and cloud database
- **Provider**: State management
- **Material Design**: UI components
